大学のHP

概要

大学の公式ホームページ制作プロジェクト。受験生・保護者・在学生・卒業生など、多様なユーザーが情報を探しやすいサイト設計が要件でした。

大規模で複雑なコンテンツ階層を整理し、明確な情報アーキテクチャ（IA）に基づいた設計。WordPress と PHP による柔軟で保守性の高いシステム構築。

担当

情報アーキテクチャ設計

ユーザー調査に基づき、学科情報・入試情報・学生生活など主要カテゴリを整理。ナビゲーション設計。

WordPress 実装 / PHP カスタマイズ

複雑なコンテンツ構造に対応。独自プラグイン・カスタム投稿タイプで柔軟性を確保。

使用技術

WordPress CMS

大規模サイト管理。複数管理者による更新運用に対応。

PHP カスタマイズ

大学独自のビジネスロジックに対応するカスタム機能を PHP で実装。

SEO 最適化

検索エンジン対応。メタタグ・サイトマップ・内部リンク設計に注力。

アクセシビリティ

WCAG ガイドラインに沿った実装。多様なユーザーへの対応。

デザインのポイント

1. 情報整理の徹底

受験生・在学生など、ユーザー属性ごとのナビゲーション。目的の情報へ素早くアクセス。

2. 視覚的階層化

カラフルすぎず、信頼感と学風を伝える配色。セクション間の明確な区切り。

3. スマートフォン対応

スマートフォンからの進学情報検索が多いため、モバイルを最優先に設計。

4. 更新運用サポート

管理画面の使いやすさ。ニュース更新・イベント掲載が容易な構造。
