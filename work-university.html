大学のHP

概要

大学の公式サイトをリニューアルするプロジェクトでした。

受験生・保護者・在学生・卒業生など、幅広い方が迷わず必要な情報にたどり着けるよう、わかりやすい導線を意識して制作しています。

たくさんの情報量があるサイトだったため、まずは内容を丁寧に整理し、見やすく探しやすい構成へと整えていきました。WordPress を使って、更新しやすい仕組みづくりにも力を入れています

担当

情報整理・サイト構成づくり

ユーザーごとに必要な情報が違うため、「受験情報」「学科紹介」「学生生活」など、大きなカテゴリからわかりやすく整理しました。迷わず目的のページに進めるよう、ナビゲーションも工夫しています。

WordPress 実装

大学ならではの多いページ数に対応できるよう、更新しやすい仕組みを WordPress で構築。必要に応じてカスタム投稿タイプも設定し、管理画面からスムーズに情報を追加できるようにしました。

使用技術

WordPress CMS

大規模なサイトでも運用しやすいよう、複数の担当者が安心して更新しやすい構造を意識して作っています。

PHP カスタマイズ

大学特有の情報構造に合わせて、必要な機能を追加しています。

SEO

検索から情報が見つけやすいよう、メタ設定やサイトマップなど、基本的な対策を整えています。

アクセシビリティ

年齢や状況を問わず、誰にとっても読みやすいサイトを目指してデザインと実装を行いました。

デザインのポイント

1. 情報をすっきり整理

ユーザーごとに知りたい情報が違うため、目的のページにスムーズに行けるよう導線を工夫しています。

2. 落ち着いたデザイン

大学らしい落ち着いた雰囲気を大事にしつつ、読みやすいようにセクションを分けて整理しました。

3. モバイルを優先した設計

進学情報はスマートフォンから検索されることが多いため、スマホでの使いやすさを最優先でデザインしています。

4. 更新しやすい仕組みづくり

「ニュースの追加」「イベント情報の掲載」など、担当者の方が負担なく更新できるよう、管理画面の操作性にも気を配りました。
